# Dio_KlabinSQL_DesafioProjeto_DashBoardXbox
Projeto desenvolvido para o desafio da DIO com o objetivo de criar um Dashboard de Vendas no Excel, transformando dados brutos em informações visuais estratégicas. O painel foi construído com tema Xbox, utilizando tabelas e gráficos dinâmicos para análise interativa e apoio à tomada de decisões.
